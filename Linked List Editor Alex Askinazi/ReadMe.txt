Linked List CSV-Editor for CSV-files.

This program is designed to edit (CRUD-principle) a "list of names"
using the Linked List method.

The "list of names" is located in the csv-file (CSVEditor.csv)
The "list of names" is also read from this file.

Linked List "list of names" saves 
to the csv-file (CSVEditor.csv) and to the txt-file (CSVEditor-txt.txt)

The main features are presented in the program menu:

1. Add Item to End
2. Add Item to Head
3. Delete Item
4. Count Items
5. Print Items 
6. Find Item (Searching)
7. Save Items to txt-file
8. Delete First (Head) Item
9. Delete Last Item
